## How To Use

```bash
# Clone this repository
$ git clone https://github.com/AdGroot800/schedule.git

# Go into the repository
$ cd schedule

# Install dependencies
$ composer install

# Run the app
$ php bin/console app:generate-quarterly-schedule
```
